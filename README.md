  <p>
    <strong>
      Halvora tracks the total 32 halvings that will occur to the Bitcoin
      protocol. It tracks block height precision, time estimates of future
      halvings and provides trading chart analysis of halving periods. The
      program's main language is Rust and uses the ICED GUI.
    </strong>
  </p>
  <p>
    <em>
      Exchange trading data source: BitStamp by Robinhood.
      Blockchain data: Mempool.space.
    </em>
  </p>
  <p>
    The exchange data for BTC is stored as daily candles going back to ~2012.
    The program handles fetch and local storage of the data with an SQLite database.
  </p>
